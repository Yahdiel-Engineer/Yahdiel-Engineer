#### 🌍 [Plateforme Numérique du Commerce Local — Centrafrique](https://github.com/Plateforme-commerce-local/api-django) *(En cours)*
> *Full-Stack Mobile + Backend + DevOps — Mon projet le plus ambitieux*

Application mobile de digitalisation du commerce informel à Bangui, RCA. Objectif : connecter commerçants et clients dans une zone où 90% des commerces n'ont aucune présence digitale.

**Ce que j'ai construit de A à Z :**
* 📱 **App Flutter** — Navigation 4 onglets, carte OpenStreetMap avec pins géolocalisés, authentification JWT, espace commerçant avec dashboard de stats
* ⚙️ **API Django REST** — 12 tables PostgreSQL + PostGIS, recherche géographique ST_DWithin, tracking analytics, signal auto-recalcul des notes
* 🐳 **Infrastructure Docker** — 3 VMs Ubuntu (dev/staging/prod), Nginx reverse proxy, pipeline CI/CD GitHub Actions
* 🗄️ **Modèle de données** — 4 phases : MVP → Paiement mobile → Logistique → Scoring crédit FinTech

**Stack :** Flutter · Dart · Django · PostgreSQL · PostGIS · Docker · Nginx · GitHub Actions · OpenStreetMap · JWT · Riverpod

```
Repo backend  → github.com/Plateforme-commerce-local/api-django
Repo frontend → github.com/Plateforme-commerce-local/app-flutter
```
